# SAP-1-Malvino-Brown
Super desafio, mas apresento-lhe minha versão do icônico SAP-1 dos livro Digital Computer Electronics de A. P. Malvino &amp; J. A. Brown (1993). O hardware foi simulado no LogiSim 2.7.1 for Windows. Também disponibilizo alguns códigos (imagem de memória) que devem ser carregados diretamente na RAM (simulada). Acrescentei as instruções JMP e STA.
